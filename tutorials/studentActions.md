# EngPot English - Student Actions

## Student Page
In Student Page, you can display:

```json
    {
        "name": "Sertan",
        "surname": "Soydabas",
        "nextLessonDate": "04-08-2022",
        "nextLessonTime": "10: 00",
        "currentCredits": "1740",

        "previousLessons": "left toggle button",
        "assignments": "right toggle button",

        "lessonsTaken": "6",
        "speakingLessonsTaken": "2",
        "lessonsCancelled": "1",
        "lessonsPostponed": "0",
        "engPotTokens": "0",
    }
```
